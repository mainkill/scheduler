[백엔드 - server/]
- 신규 폴더: server/
- 기술 스택: Node.js + Express + Prisma ORM + SQLite
- 패키지: express, @prisma/client, bcryptjs, jsonwebtoken, cors, dotenv
- Prisma 스키마: server/prisma/schema.prisma
  - User 모델 (id, email, name, password, createdAt)
  - Task 모델 (id, title, dueDate, priority, status, createdAt, updatedAt, userId)
- DB 마이그레이션: server/prisma/dev.db (SQLite) 생성 완료
- JWT Secret: 128자 랜덤 생성 후 server/.env에 저장
- API 엔드포인트:
  - POST /api/auth/register  (회원가입)
  - POST /api/auth/login     (로그인)
  - GET    /api/tasks        (일정 목록 조회)
  - POST   /api/tasks        (일정 생성)
  - PUT    /api/tasks/:id    (일정 수정)
  - PATCH  /api/tasks/:id/toggle (완료 토글)
  - DELETE /api/tasks/:id    (일정 삭제)
- 인증 방식: JWT Bearer Token (7일 만료)
- 비밀번호: bcrypt (salt rounds 12)
